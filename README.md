# Fr3oon Helper Bot

بوت حماية لجروب تبادل بطاقات MOBA SHOP.

## بيعمل ايه؟

- يمسح اي لينك
- يمسح اي رسالة مش صورة
- يسمح بصور بطاقات ببجي
- يمنع نفس الصورة لو اتبعت قبل كده خلال 3 ايام
- يسيب رسائل الادمنز عادي

## التشغيل على Koyeb

1. ارفع الملفات دي على GitHub.
2. اعمل Deploy على Koyeb كـ Web Service.
3. ضيف Environment Variable باسم:

```text
BOT_TOKEN
```

وحط فيه توكن البوت الجديد من BotFather.

مهم: متحطش التوكن جوه الكود ولا على GitHub.

## صلاحيات البوت في الجروب

خلي البوت Admin وفعل له:

```text
Delete messages
```

بس كده.
